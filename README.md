# Интернет-магазин электронной техники

## Описание проекта
Фронтенд приложение интернет-магазина электронной техники, разработанное на TypeScript с применением принципов ООП и паттернов проектирования GoF.

## Архитектурные принципы
- **SOLID** - все принципы соблюдены в проектировании системы
- **TypeScript** - строгая типизация для повышения надежности кода
- **GoF Patterns** - все 23 паттерна применены в соответствующих контекстах

## Функциональные модули

### 1. Каталог товаров
- Древовидная структура категорий (Composite)
- Итерация по товарам (Iterator)
- Ленивая загрузка изображений (Proxy)
- Кэширование данных товаров (Flyweight)
- Статистика просмотров (Visitor)

### 2. Корзина покупок
- Отмена/повтор действий (Command, Memento)
- Уведомления об изменениях (Observer)
- Состояния корзины (State)
- Различные стратегии расчета (Strategy)

### 3. Система оплаты
- Поддержка разных платежных систем (Strategy, Abstract Factory)
- Адаптация к внешним API (Adapter)
- Абстракция платежных методов (Bridge)
- Защищенные запросы (Proxy)

### 4. Управление пользователями
- Единая точка доступа к сервису (Singleton)
- Создание различных типов пользователей (Factory Method)
- Клонирование профилей (Prototype)
- Централизованное управление зависимостями (Mediator)

## Технические особенности
- Полное покрытие TypeScript типами
- Модульная архитектура с слабой связностью
- Переиспользуемые компоненты
- Расширяемая система плагинов и стратегий

## Паттерны проектирования
Все 23 паттерна из книги "Банды четырех" применены в системе согласно принципу "правильный инструмент для правильной задачи".
